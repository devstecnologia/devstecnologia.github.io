1. Introdução

O ChurrasCalc, publicado por Devs T I LTDA, auxilia no planejamento de churrascos por meio de cálculos, checklist e histórico. Esta política explica como tratamos seus dados em conformidade com a LGPD.

Nossa Promessa

Minimizar a coleta, manter dados locais sempre que possível e ser transparentes sobre permissões e compartilhamentos.

2. Dados Coletados

2.1 Informações que Você Fornece

  • Dados do App: Itens do checklist, configurações e históricos inseridos
  • Preferências: Configurações personalizadas do aplicativo

2.2 Informações Coletadas Automaticamente

  • Armazenamento Local: Dados salvos no dispositivo usando GetStorage
  • Informações do Dispositivo: Modelo, sistema operacional, versão do app

Importante: Não coletamos dados pessoais automaticamente. Não utilizamos analytics por padrão.

3. Como Usamos seus Dados

3.1 Finalidades Principais

  • Salvar preferências e conteúdo do checklist/histórico no armazenamento local do dispositivo
  • Gerar conteúdo para compartilhamento quando você aciona o recurso de compartilhar
  • Fornecer funcionalidades offline do aplicativo

3.2 Tecnologias Utilizadas

  • GetStorage: Para armazenamento local seguro de dados
  • Share Plus: Para compartilhamento de conteúdo com outros apps

Base Legal

Utilizamos seus dados com base no legítimo interesse, execução de contrato, cumprimento de obrigação legal ou seu consentimento explícito, conforme aplicável.

4. Compartilhamento de Dados

Não vendemos, alugamos ou comercializamos seus dados pessoais. Podemos compartilhar informações apenas nas seguintes situações:

4.1 Compartilhamento Voluntário

  • Quando você opta por compartilhar conteúdo através de outros aplicativos instalados
  • Apenas o conteúdo que você escolher compartilhar será enviado

4.2 Prestadores de Serviços

  • Bibliotecas Flutter para funcionalidades básicas do app
  • Serviços de compartilhamento nativos do sistema operacional

Garantia: Seus dados pessoais nunca são compartilhados sem seu consentimento explícito.

5. Segurança dos Dados

Implementamos medidas técnicas e organizacionais apropriadas para proteger seus dados:

  • Armazenamento Local: Dados ficam armazenados apenas no seu dispositivo
  • GetStorage Seguro: Utilizamos armazenamento criptografado local
  • Sem Sincronização: Não há transmissão de dados para servidores externos
  • Boas Práticas: Desenvolvimento seguindo padrões de segurança

Recomendação Importante

Recomendamos não inserir informações sensíveis no app, pois os dados ficam armazenados no próprio dispositivo e podem ser acessados por outras pessoas com acesso físico ao aparelho.

6. Seus Direitos

De acordo com a LGPD, você tem os seguintes direitos:

  • Acesso: Solicitar informações sobre o tratamento dos seus dados
  • Correção: Corrigir dados incompletos, inexatos ou desatualizados
  • Exclusão: Solicitar a eliminação dos dados desnecessários
  • Portabilidade: Solicitar a portabilidade dos dados
  • Oposição: Opor-se ao tratamento baseado em legítimo interesse
  • Revogação: Revogar o consentimento a qualquer momento

Como Exercer seus Direitos

Para exercer qualquer um desses direitos, entre em contato conosco através do email devstecnologia@gmail.com.

Nota: Como os dados ficam armazenados localmente no dispositivo, você também pode gerenciá-los diretamente através das configurações do sistema operacional.

7. Cookies e Tecnologias Similares

7.1 Tecnologias Utilizadas

  • GetStorage: Para armazenamento local seguro de preferências e dados
  • Share Plus: Para compartilhamento de conteúdo com outros apps
  • Flutter Libraries: Bibliotecas básicas para funcionamento do app

7.2 Permissões do App

  • Internet: Utilizada por bibliotecas do Flutter e para compartilhar conteúdo
  • PROCESS_TEXT (Android): Permite interagir com apps de processamento de texto
  • Sem acesso: Não requer localização, câmera, microfone ou contatos

Importante: Não há sincronização com servidores próprios e não coletamos dados desses apps.

8. Alterações na Política

Esta política pode ser atualizada periodicamente para refletir mudanças em nossas práticas ou requisitos legais. Quando isso ocorrer:

  • Publicaremos a versão atualizada nesta página
  • Atualizaremos a data da última modificação
  • Notificaremos sobre mudanças significativas quando possível

Recomendamos que revise esta política periodicamente para se manter informado sobre como protegemos suas informações.
